Modern WordPress Integration

WP-Node lets you build modern applications on top of WordPress databases using TypeScript and Node.js.

Type-Safe & Minimal

Designed with type safety and simplicity in mind, WP-Node uses Knex.js for query building and Zod for schema validation.

Powerful CLI & Configurability

From seeding WordPress-compatible tables to running custom CLI tasks, WP-Node provides tools to bootstrap and scale development quickly.
